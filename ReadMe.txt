1. To play this game you need to have installed python and python libraries pygame and pygame_menu

---------------------------follow this steps-----------------------------------------------------

first install latest version of python3 also add pip feature in PATH.
open command prompt then type "pip install pygame" and "pip install pygame_menu"
and thats all
Now go to Space Fighter folder and open cmd in that directory 
now type "py play.py" start playing.....
